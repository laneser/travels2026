# 研究來源說明

本趟（202607Korea）的原始研究由 multi-agent workflow `seoul-day6-research` 產出，
8 個平行 agent 各自負責一個面向，透過 WebSearch / WebFetch 查證後彙整：

1. 正餐韓式美食（烤肉、湯飯鍋物、蔘雞湯、炸雞、冷麵）
2. 傳統市場與街頭小吃（廣藏市場、東大門夜市、明洞小吃等）
3. 咖啡廳與甜點冰品（聖水洞、益善洞、安國／北村、弘大）
4. 景點（古宮文化、東大門／明洞、弘大、聖水、漢江／首爾塔、蠶室／松坡）
5. 購物（東大門、明洞、弘大、聖水、K-pop 周邊）
6. 交通（GMP↔東大門、地鐵、T-money、KSPO DOME 場館交通、回程）
7. 演唱會 logistics（DAY6〈The DECADE〉FINALE、手燈、換票、周邊、散場）
8. 天氣與實用（7 月梅雨季、打包、現金／退稅／App、雨天備案）

彙整後的結構化內容見 `research.md`。實際採用並排進行程的版本，
以 `../../src/data.js`（SSOT）為準——兩者若有出入，以 `data.js` 為對。

⚠️ 研究中的營業時間、價位、票務規定為「規劃當時」的快照，
出發前請務必再以官方公告 / Naver Map 確認（尤其演唱會換票與入場規定）。
