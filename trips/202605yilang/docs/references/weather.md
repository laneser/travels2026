# 宜蘭五月天氣（凍結於 2026-04-21）

## 特徵
- **梅雨季**（5–6 月為台灣梅雨主力期）
- 氣溫高溫 27–29°C、低溫 20–22°C
- 降雨機率高、降雨日多
- 濕度高，體感偏悶

## 打包重點
- **雨傘／輕便雨衣必帶**
- 快乾衣物（梅雨濕氣重）
- 薄外套（早晚溫差 + 室內冷氣強）
- 防蚊液（宜蘭濕熱蚊蟲多）
- 夾腳拖／防水涼鞋（逛夜市雨天用）

## 雨天備案
- 蘭陽博物館（室內）
- 傳藝中心（多為室內展館）
- 奇麗灣珍奶文化館（DIY + 室內）
- 羅東林業文化園區（可撐傘散步）
- 礁溪溫泉（雨天泡湯最舒服）
- 室內烤肉民宿（若有訂則免出門也 OK）
