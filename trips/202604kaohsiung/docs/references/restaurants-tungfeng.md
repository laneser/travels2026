## 抓取方式（for future reference）

```bash
# 搜尋痛風老饕 高雄 相關影片
node ~/.claude/skills/youtube-survey/scripts/search.mjs "痛風老饕 高雄" --max 10 --out videos.json

# 抓一支影片的描述 + 高讚留言（過濾有時間戳+地址的留言）
node ~/.claude/skills/youtube-survey/scripts/scrape.mjs 3QK6nFhTpoo \
  --comments 40 \
  --filter "(\\d{1,2}:\\d{2}|高雄市)" \
  --out video.json
```

純 HTTP，不需 browser、不需 API key；呼叫的是 YouTube 網站自己用的 InnerTube API。
